## 任务拆解
- 图片处理：删除你截图中标注的具体区域（而非整张图片），边缘自然过渡、分辨率不变，并提供删除前后对比预览
- 导航恢复：在首页与全站恢复并固定顶部导航栏，保持选中态与悬停态、点击区域充足
- 新增首页“记录学习”板块：在主页中加入轻量笔记输入与列表入口，后续可独立为 /notes

## 图片编辑实现
- 页面：新增 `/image-editor`（仅实现你给的图片处理需求，支持后续扩展）
- 功能与交互：
  - 上传图片（支持你给的链接或本地）
  - 选择删除区域（矩形/套索/笔刷）
  - 删除并填充：
    - 基线：邻域颜色插值 + 羽化边缘（feather）保证自然过渡
    - 可选增强：调用在线 Inpainting API（需你的 API Key）以获得更真实的“内容感知填充”
  - 对比预览：左右分屏、拖动分隔条查看前后效果
  - 导出：保留原分辨率与质量
- 性能：大图分块渲染 + Web Worker 处理像素计算，避免主线程卡顿

## 主页导航恢复与优化
- 全局导航在 `layout` 顶部固定（sticky top-0 z-40）
- 选中态/悬停态/点击区域：
  - 当前路由高亮（加粗+底边线/浅背景）
  - 悬停 `hover:bg-slate-100`、可点击区域 `px-3 py-2`
- 移动端抽屉菜单，保持所有页面一致
- 平滑过渡：轻微淡入，不影响路由性能

## 首页“记录学习”板块
- 位置：Hero 下方新增“记录学习”卡片区，提供输入框与快捷格式工具（加粗/斜体/下划线/H1/H2/标色）
- 列表：最近记录的卡片展示（标题/摘要/标签/颜色条），点击进入完整记录界面
